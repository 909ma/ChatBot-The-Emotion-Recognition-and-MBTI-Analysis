#### 개선점

    MBTI 데이터 추출에 사용된 자료의 출처가 디시 인 사이드이기에 질적으로 충분하지 못한 데이터입니다. 그 외에도 속도 문제로 인하여 카카오 채널 봇도 퍼블리싱했으나 응답시간의 지연으로 인해 정상적으로 사용하지 못합니다.  
  

    추가로 한국어로 된 말뭉치 데이터 중 사용하기 좋게 가공된 것이 없으므로 시간이 더 필요합니다.
